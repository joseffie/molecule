## ‍💼 «Molecule»
Landing page for SaaS marketing company

## 🖥️ Deployment
To deploy this project run

```
$ npm install
$ npm run dev
```

## 🛠 Skills
- Pug.
- Sass (SCSS syntax).
- JavaScript.

## 🛠 Features
- Using WebP images.
- Using Grid CSS.
- Using Swiper for slider.
- Local fonts.
